# Modela-de-casos-de-uso-UML-
Ejercicios de modelado de casos de uso TAREA


## 📌 Descripción del Proyecto

Este repositorio contiene el modelado de **diagramas de casos de uso UML** desarrollados como parte del estudio de **Programación Orientada a Objetos**.

El objetivo principal del proyecto es representar gráficamente distintos procesos reales mediante diagramas UML, identificando correctamente **actores**, **casos de uso**, y relaciones como **<<include>>** y **<<extend>>**.

Los diagramas representan diferentes escenarios de sistemas automatizados, permitiendo comprender la interacción entre los usuarios y los sistemas.

## Tecnologías Utilizadas

Las siguientes herramientas fueron utilizadas para el desarrollo del proyecto:

* **UML (Unified Modeling Language)**
  Para el modelado de diagramas de casos de uso.

* **Lucidchart**
  Herramienta utilizada para la creación y diseño visual de los diagramas UML.

* **GitHub**
  Para el control de versiones y almacenamiento del repositorio.
* **ChatGPT**
  Obtención de datos.

## Objetivos del Proyecto

* Aplicar conceptos fundamentales de UML.
* Identificar correctamente actores y casos de uso.
* Utilizar relaciones **<<include>>** y **<<extend>>**.
* Modelar sistemas reales mediante diagramas de casos de uso.
* Documentar el trabajo utilizando buenas prácticas en GitHub.
